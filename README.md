# Simple Gulp Builder
Include plugins :
  - Autoprefixer
  - Css-mqpacker
  - Csswring
  - Gulp-concat
  - Gulp-file-include
  - Gulp-less-sourcemap
  - Gulp-postcss
  - Gulp-sourcemaps
  - Gulp-uglify
  - Gulp-watch
  - Gulp-watch-less

Как начать:
HTML в корне проекта собирается через dev/templates + dev/chunks.
Конструкция "@@include('../chunks/head.html')" -> ищет данный темплейт в chunks.

Для запуска - открываем консоль в нашем проекте, пишем - gulp .
